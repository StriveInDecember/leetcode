## 102.二叉树的后序遍历 

给你一个二叉树，请你返回其按 层序遍历 得到的节点值。 （即逐层地，从左到右访问所有节点）。

- 常规思路
```python
    class Solution:
        def levelOrder(self, root: TreeNode) -> List[List[int]]:
            res = list()
            if not root: return res
            stack = [root]
            while stack:
                memo = list()
                tmp = list()
                for node in stack:
                    tmp.append(node.val)
                    if node.left: memo.append(node.left)
                    if node.right: memo.append(node.right)
                res.append(tmp)
                stack = memo
            return res
    ```
