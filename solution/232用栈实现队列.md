## 232. 用栈实现队列

请你仅使用两个栈实现先入先出队列。队列应当支持一般队列支持的所有操作（push、pop、peek、empty）：

```text
实现 MyQueue 类：
void push(int x) 将元素 x 推到队列的末尾
int pop() 从队列的开头移除并返回元素
int peek() 返回队列开头的元素
boolean empty() 如果队列为空，返回 true ；否则，返回 false
```

特点： 栈 - 后进先出  队列 - 先进先出

- 常规思路：一个list解决
    ```python
    class MyQueue:
    
        def __init__(self):
            self.stack = list()
    
        def push(self, x: int) -> None:
            self.stack.append(x)
    
        def pop(self) -> int:
            return self.stack.pop(0)
    
        def peek(self) -> int:
            return self.stack[0]
    
        def empty(self) -> bool:
            return not self.stack
    ```
- 按照题意，两个栈实现。
  - 一种思路是，一个栈里面放置排序好的所有元素，另一个栈用来辅助排序。
    ```python
    class MyQueue:
    
        def __init__(self):
            self.stack1 = list()
            self.stack2 = list()
    
        def push(self, x: int) -> None:
            while self.stack1:
                self.stack2.append(self.stack1.pop())
            self.stack1.append(x)
            while self.stack2:
                self.stack1.append(self.stack2.pop())
    
        def pop(self) -> int:
            return self.stack1.pop() if self.stack1 else None
    
        def peek(self) -> int:
            return self.stack1[-1]
    
        def empty(self) -> bool:
            return not self.stack1
    ```
  - 一个栈作为压入栈，一个栈作为弹出栈。判断是否为空时，两个都需要判断
    ```python
    class MyQueue:
    
        def __init__(self):
            self.stack1 = list()
            self.stack2 = list()
    
        def push(self, x: int) -> None:
            self.stack1.append(x)
    
        def pop(self) -> int:
            if not self.stack2:
                while self.stack1:
                    self.stack2.append(self.stack1.pop())
            return self.stack2.pop()
    
        def peek(self) -> int:
            if not self.stack2:
                while self.stack1:
                    self.stack2.append(self.stack1.pop())
            return self.stack2[-1]
    
        def empty(self) -> bool:
            return not self.stack1 and not self.stack2
    ```
